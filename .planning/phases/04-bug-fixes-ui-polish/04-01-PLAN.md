Audit each screen against the design system checklist from 04-RESEARCH.md:

For each screen, check:
1. **Loading State**: Uses VlvtLoader during initial load? No blank screen flash?
2. **Error State**: Catches all async errors? Uses ErrorHandler? Shows retry option?
3. **Empty State**: Lists handle empty case? Uses EmptyStateWidget or custom variant?
4. **Design Consistency**: Colors from VlvtColors? Typography from VlvtTextStyles? Spacing from Spacing constants? Buttons use VlvtButton? Inputs use VlvtInput?
5. **Navigation**: All tappable elements work? Back navigation works? No dead ends?
6. **Edge Cases**: Offline mode handled? Long text truncates? Missing images show placeholder?
